Použij angličtinu!

  1. JavaScript is the programming language of the Web
  2. JavaScript is the default scripting language in HTML
  3. You can place any number of scripts in an HTML document
  4. JavaScript code is inserted between script and /script tags

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Připojit externí dokument
<script src="myScript1.js"></script>
// 3. Připojit odkaz na web, kde je uložený soubor s JS
<script src="https://www.w3schools.com/js/myScript1.js"></script>
// 4. Připojit cestu na soubor s JS
<script src=""/js/myScript1.js""></script>

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // s prázdnou hodnotou
let x = 18; // celé číslo (typ number)
let x = 17.5;// reálné číslo (typ number)
let x = 123e-2; // zápis desetinného čísla pomocí exponentu
let x = "Vojta"; // typ string
let prestupny = false; // typ boolean(true/false)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

= //přiřazení
+= //přičtení, ale také připojení řetězce
*=, -=, /= //přinásobení, odečtení, "přidělení"
++ //přičtení 1
-- //odečtení 1

Logické

&& // logické AND (a zároveň)
|| //logické OR (nebo)
! // logické NOT (negace)
, // logické spojení (třeba v zápisu parametrů funkcí)

Porovnávací

== //rovnost (dvě rovnítka)
=== //rovnost hodnoty i typu
!= //nerovnost
!== //nerovnost hodnoty i typu
? : //podmínkový výběr (ternární operátor)
<,<=,>=,> // aritmetické srovnání

Bitové

<< //bitový posun vlevo
>> //bitový posun vpravo
& //bitový součin
| //bitový součet
~ //bitová negace
^ //bitová nonekvivalence

Typové

typeof (typ proměnné), instanceof (vrací true, pokud je objekt součástí daného objektu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Při vyvolání vyskočí okno s daným textem (Nezadal jsi správné heslo!) */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt("Zadej heslo.");

confirm()

/* Vyskočí okno s nabídkou Cancel/OK. Při kliknutí na Ok bude uživatel přenesen na danou stránku (Seznam.cz). */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
Vyskakovací okna často obsahují nežádoucí reklamy.
Zároveň jsou rušivé, jelikož při jejich vyskočení nemůžeme "brouzdat" po stránce.
